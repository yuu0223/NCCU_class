### Course
> **軟體工程 Software Engineering**

**簡介**
- 學習軟體開發所具備之相關知識，像是軟體生命週期、敏捷開發等。
- 學習軟體開發所需之工具使用，像是GitLab版本控制、CICD等。

**使用工具**
- Python (FastAPI)：後端開發
- MySQL：Local Database
- Docker：讓大家使用相同的環境，較不會出現衝突
- Google sheet：功能規劃需求表
- GitLab
